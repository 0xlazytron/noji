### Installation
> **Warning**
> Be careful running this with elevated privileges. Code execution can be achieved with write permissions on the config file.

#### Method 2: Install From Source

```bash
$ git clone https://www.github.com/0xlazytron/noji.git
$ yarn add all
$ yarn start
```
